URL Shortener Microservice

Make a valid GET request to the following url:

https://url-tin.herokuapp.com/new/

followed by a valid URL like so:

https://url-tin.herokuapp.com/new/http://foo.com:80

and it will return a JSON object including the appended URL and the shortened one. Like so:

{"original_url": "http://foo.com:80", "short_url": "https://url-tin.herokuapp.com/*******"}
